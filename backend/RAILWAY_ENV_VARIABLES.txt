🎯 DEPLOYMENT STEPS:

1. Add MySQL Database Addon:
   Railway Project → New → Database → MySQL

2. Set User Service Variables:
   User Service → Settings → Variables → RAW Editor
   → Paste variables from above
   → Click "Add" or "Update"

3. Set Submission Service Variables:
   Submission Service → Settings → Variables → RAW Editor
   → Paste variables from above
   → Click "Add" or "Update"

4. Add Volume for File Uploads (Submission Service only):
   Submission Service → Settings → Volumes
   → Mount Path: /app/uploads
   → Click "Add Volume"

5. Update CORS_ALLOWED_ORIGINS:
   Replace with your actual frontend URL in BOTH services

6. Deploy:
   Both services will auto-deploy when you push to GitHub
   OR click "Deploy" button in Railway dashboard

7. Verify:
   User Service: https://your-user-service.up.railway.app/actuator/health
   Submission Service: https://your-submission-service.up.railway.app/actuator/health
   Both should return: {"status":"UP"}
